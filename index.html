DictionaryApp

This web application integrates Firebase for user authentication and a Dictionary API to enable word exploration and management. Firebase handles user login, registration, and logout, updating the UI based on authentication status. Users can add words to favorites, stored locally with options to remove them. The app tracks and displays search history for quick access to previous searches. Features include audio playback for word pronunciations, error handling for API requests, and redirects upon successful login/registration.

Technologies used: HTML, Bootstrap, CSS,JavaScript,Firebase,Dictionary API (https://api.dictionaryapi.dev)

QuizWiz

This web application is a quiz platform that fetches multiple-choice questions from the Open Trivia Database API based on user-selected criteria such as the number of questions, category, difficulty, and type (multiple-choice or true/false). Users can start the quiz, answer the questions within a set time limit (30 seconds per question), and submit their answers. Upon submission, the application evaluates the answers, displays the score, reveals correct answers, and disables further interaction with the questions

Technologies used: HTML, CSS, JavaScript,Opentdb API
